# Selene 音乐播放器

一个可通过 Git 仓库链接直接安装的 SillyTavern 第三方音乐播放器扩展。

## 功能

- 搜索和播放网络音乐
- 歌词显示与桌面悬浮歌词
- 播放队列与歌曲收藏
- 导入网易云、QQ 音乐或酷狗歌单
- 读取世界书中的作者推荐歌单
- 自动读取 AI 回复中的歌曲推荐
- 多种皮肤、迷你模式与沉浸模式
- 播放器和歌词尺寸自定义
- 系统媒体保活
- 桌面端与移动端适配

## 安装

1. 打开 SillyTavern，点击顶部的「扩展」。
2. 选择「安装扩展」。
3. 粘贴本仓库的 HTTPS 链接：

   ```text
   https://github.com/soso454u/-
   ```

4. 点击安装，安装完成后会立即加载并显示播放器，不需要手动刷新。若不成功按照 SillyTavern 的提示重新加载页面。

> 请不要粘贴 `raw.githubusercontent.com` 文件链接、ZIP 下载链接或单个 JS 文件链接。SillyTavern 需要的是完整 Git 仓库链接。

## 使用说明

### 顶部按钮

- **×**：关闭播放器，之后可以从 SillyTavern 的扩展菜单重新打开。
- **标题「Selene Music」**：点击缩成迷你封面，再点迷你封面恢复；长按可以拖动播放器。
- mini 悬浮封面不会显示关闭叉；点击 mini 封面会恢复播放器，不会误关闭。
- **标 / 小 / 大**：切换播放器内容和按钮的大小。
- **词**：开启或关闭桌面悬浮歌词。
- **📌**：固定播放器，防止误拖动或缩放。
- **✦**：切换播放器皮肤。
- **⚙**：打开详细设置。

### 找歌与播放

- **搜索框**：输入歌曲名或歌手，点击搜索。
- **读取本层推荐**：读取当前楼层 AI 回复中的歌曲推荐，并自动搜索。
- **读取推荐**：读取当前角色或已导入世界书中的作者推荐歌单。
- **点击歌曲整行**：立即播放。
- **☆**：收藏或取消收藏歌曲。
- **＋**：将歌曲加入播放队列。

### 收藏与播放队列

- **收藏**：查看已经保存的歌曲。
- **收藏入队**：将全部收藏歌曲加入播放队列。
- **一键播放**：从第一首开始连续播放收藏歌曲。
- **队列**：查看接下来要播放的歌曲，数字代表当前队列数量。
- **×**：从收藏或播放队列中删除歌曲。
- 队列歌曲旁的 **☆**：将这首歌保存到收藏。

### 播放模式

点击播放模式按钮可以循环切换：

- **循环 ↻**：按照顺序循环播放。
- **单曲 ↺**：重复播放当前歌曲。
- **随机 ⤨**：随机播放歌曲。

### 歌词

- **歌词 🔓**：歌词可以拖动。
- **歌词 🔒**：锁定歌词位置，防止误触。
- **词**：快速开启或关闭桌面悬浮歌词。
- 手机版可以在设置中单独开启悬浮歌词。
- 点击歌曲封面可以进入大封面沉浸模式，再次点击封面或右上角箭头即可返回。

### 播放控制

- **‹‹**：上一首。
- **-10**：后退 10 秒。
- **▶ / ❚❚**：播放或暂停。
- **+10**：前进 10 秒。
- **››**：下一首。
- **进度条**：拖动跳转到歌曲的指定位置。
- **音量条**：调节播放器音量。

### 导入歌单

点击 **导入歌单**，粘贴支持的歌单链接：

- 网易云音乐歌单链接
- QQ 音乐歌单链接
- 酷狗音乐歌单链接

导入成功的歌曲会自动保存到播放器收藏中。

### 世界书作者推荐歌单

仓库提供了可直接导入 SillyTavern 的 [`Selene作者推荐歌单-世界书模板.json`](./Selene作者推荐歌单-世界书模板.json)。导入后可将世界书绑定给角色，也可以作为普通世界书保存；点击播放器中 **导入歌单** 后面的 **读取推荐** 即可读取。

世界书条目可以保持禁用，避免歌单内容进入聊天提示词。播放器会读取 `<selene_playlist>` 专用标签，不会把世界书的普通正文误识别成歌曲。

作者只需按下面的格式填写歌名和歌手，每首歌占两行：

```text
<selene_playlist>
歌单名：角色名的推荐歌单

歌曲：Too Much
歌手：Dove Cameron

歌曲：微醺夜
歌手：Veena
</selene_playlist>
```

也支持一行简写：

```text
<selene_playlist>
Too Much | Dove Cameron
微醺夜 | Veena
</selene_playlist>
```

请保留开头与结尾的 `<selene_playlist>` 标签。读取成功后，可以逐首播放、收藏或加入队列，也可以点击 **全部入队** / **一键播放**。

### 设置里的功能

- **自动读取推荐**：AI 回复中出现歌曲推荐时，自动搜索并播放。
- **系统媒体保活**：未选中任何歌曲时运行静音媒体；选中歌曲后，播放和暂停状态都由歌曲独占系统媒体卡片，避免静音保活抢走播放按钮。扩展版使用内置 AAC，脚本版使用兼容的 44.1kHz WAV。
- **回到保活**：扩展设置栏中的手动切换按钮。点击后暂停当前歌曲并让静音媒体接管；下次从播放器播放歌曲时，静音保活会自动让位。
- **主题**：切换跟随系统、日间或夜间模式。
- **桌面歌词 / 手机版悬浮歌词**：分别控制电脑端和手机端的歌词显示。
- **歌词主色 / 渐变副色**：调整悬浮歌词颜色。
- **独立尺寸调整**：分别修改按钮、文字、封面和歌曲列表的大小。
- **重置歌词位置**：歌词位置跑偏时恢复默认位置。
- **恢复播放器默认尺寸**：播放器窗口大小乱掉时一键恢复。

### Selene 扩展控制栏

在 SillyTavern 的扩展设置页中展开「Selene 音乐播放器」，可以：

- 显示或隐藏播放器窗口。
- 选择启动 SillyTavern 时是否自动显示播放器。
- 选择是否在扩展菜单显示「音乐播放器」入口。
- 启用或停止系统媒体保活，并查看是否已经真正开始播放。
- 在支持 Media Session 的浏览器中，锁屏或控制中心会显示当前歌曲、歌手、专辑封面、真实时长与进度，并可播放、暂停、切换歌曲或拖动进度。
- 播放器会在真实音频开始播放、封面更新及页面回到前台时重新提交歌名与歌手，减少 Chrome 控制中心只显示应用名称的问题。
- Android Chrome、Android/Windows Edge 与 iOS Safari 均可通过浏览器的 Media Session 接入系统媒体界面；首次播放必须由用户点击歌曲或播放键触发。部分安卓系统还需要允许浏览器通知、后台活动并关闭严格省电限制。
- 设置面板会直接显示当前浏览器是否检测到 `navigator.mediaSession`；这可用于检查雨见等采用不同内核的浏览器版本。
- 分别重置播放器位置与窗口尺寸。

### 系统媒体保活怎么开

可以从「Selene 音乐播放器」扩展控制栏开启，也可以在播放器中打开：

**⚙ → 系统媒体保活 → ▶ 启动保活**

开启后会在尚未选中歌曲时播放本地静音媒体。一旦选中歌曲，静音保活就会让位，即使歌曲处于暂停状态也不会抢走系统媒体卡片。状态栏的播放按钮会恢复当前歌曲。如果浏览器没有自动开始，扩展控制栏会显示原生音频播放条，点击其中的播放键即可。

显示 **系统媒体已播放** 或 **系统媒体已启动** 就代表已经开启。iPhone/iPad 切到后台后，控制中心或锁屏界面可能显示 Chrome/SillyTavern 的媒体控制；不需要时可以点击 **■ 停止保活**。

> 系统媒体保活可以降低浏览器休眠导致暂停播放或掉线的概率，但如果手机系统直接冻结或关闭浏览器，网页无法保证百分之百不断线。

## 数据来源与使用说明

- **音乐与歌词数据**：[GD 音乐台](https://music.gdstudio.xyz)
- 播放器恢复 1.1 版 GD 严格匹配与播放流程；播放失败时会自动重试 5 次。
- 手机端不再使用静音占位音源预热主播放器，避免 Safari 已载入音源却被误判为播放超时。
- 本扩展仅作为第三方播放器界面，不在仓库中储存或提供音乐文件。
- 音乐搜索、音频及歌词服务的可用性、内容与地区限制由对应第三方服务决定。
- 本项目仅供个人学习与交流使用，请勿用于商业用途。
- 使用时请遵守当地法律法规、音乐平台条款及数据来源方的使用要求。
- 如相关服务失效、接口调整或歌曲版权受限，部分歌曲可能出现无法搜索、播放或显示歌词的情况。

> 本项目与网易云音乐、QQ 音乐、酷狗音乐及 GD 音乐台均无官方隶属或合作关系。
