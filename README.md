# Selene 音乐播放器

一个可通过 Git 仓库链接直接安装的 SillyTavern 第三方音乐播放器扩展。

## 功能

- 搜索和播放网络音乐
- 歌词显示与桌面悬浮歌词
- 播放队列与歌曲收藏
- 导入网易云、QQ 音乐或酷狗歌单
- 自动读取 AI 回复中的歌曲推荐
- 多种皮肤、迷你模式与沉浸模式
- 播放器和歌词尺寸自定义
- 系统媒体保活
- 桌面端与移动端适配

## 安装

1. 确保运行 SillyTavern 的设备已安装 Git。
2. 打开 SillyTavern，点击顶部的「扩展」。
3. 选择「安装扩展」。
4. 粘贴本仓库的 HTTPS 链接：

   ```text
   https://github.com/soso454u/-
   ```

5. 点击安装，并按照 SillyTavern 的提示重新加载页面。

在支持扩展生命周期钩子的新版 SillyTavern 中，安装完成后会立即加载并显示播放器，不需要手动刷新。如果旧版 SillyTavern 显示重新加载提示，请按客户端提示操作。

> 请不要粘贴 `raw.githubusercontent.com` 文件链接、ZIP 下载链接或单个 JS 文件链接。SillyTavern 需要的是完整 Git 仓库链接。

## 使用说明

### 顶部按钮

- **×**：关闭播放器，之后可以从 SillyTavern 的扩展菜单重新打开。
- **标题「Selene Music」**：点击缩成迷你封面，再点迷你封面恢复；长按可以拖动播放器。
- **标 / 小 / 大**：切换播放器内容和按钮的大小。
- **词**：开启或关闭桌面悬浮歌词。
- **📌**：固定播放器，防止误拖动或缩放。
- **✦**：切换播放器皮肤。
- **⚙**：打开详细设置。

### 找歌与播放

- **搜索框**：输入歌曲名或歌手，点击搜索。
- **读取本层推荐**：读取当前楼层 AI 回复中的歌曲推荐，并自动搜索。
- **点击歌曲整行**：立即播放。
- **☆**：收藏或取消收藏歌曲。
- **＋**：将歌曲加入播放队列。

### 收藏与播放队列

- **收藏**：查看已经保存的歌曲。
- **收藏入队**：将全部收藏歌曲加入播放队列。
- **一键播放**：从第一首开始连续播放收藏歌曲。
- **队列**：查看接下来要播放的歌曲，数字代表当前队列数量。
- **×**：从收藏或播放队列中删除歌曲。
- 队列歌曲旁的 **☆**：将这首歌保存到收藏。

### 播放模式

点击播放模式按钮可以循环切换：

- **循环 ↻**：按照顺序循环播放。
- **单曲 ↺**：重复播放当前歌曲。
- **随机 ⤨**：随机播放歌曲。

### 歌词

- **歌词 🔓**：歌词可以拖动。
- **歌词 🔒**：锁定歌词位置，防止误触。
- **词**：快速开启或关闭桌面悬浮歌词。
- 手机版可以在设置中单独开启悬浮歌词。
- 点击歌曲封面可以进入大封面沉浸模式，再次点击封面或右上角箭头即可返回。

### 播放控制

- **‹‹**：上一首。
- **-10**：后退 10 秒。
- **▶ / ❚❚**：播放或暂停。
- **+10**：前进 10 秒。
- **››**：下一首。
- **进度条**：拖动跳转到歌曲的指定位置。
- **音量条**：调节播放器音量。

### 导入歌单

点击 **导入歌单**，粘贴支持的歌单链接：

- 网易云音乐歌单链接
- QQ 音乐歌单链接
- 酷狗音乐歌单链接

导入成功的歌曲会自动保存到播放器收藏中。

### 设置里的功能

- **自动读取推荐**：AI 回复中出现歌曲推荐时，自动搜索并播放。
- **系统媒体保活**：播放扩展内置的 10 小时本地静音 AAC，让 iOS/Chrome 建立真实的系统媒体会话，减少切到后台后被暂停的情况。
- **主题**：切换跟随系统、日间或夜间模式。
- **桌面歌词 / 手机版悬浮歌词**：分别控制电脑端和手机端的歌词显示。
- **歌词主色 / 渐变副色**：调整悬浮歌词颜色。
- **独立尺寸调整**：分别修改按钮、文字、封面和歌曲列表的大小。
- **重置歌词位置**：歌词位置跑偏时恢复默认位置。
- **恢复播放器默认尺寸**：播放器窗口大小乱掉时一键恢复。

### Selene 扩展控制栏

在 SillyTavern 的扩展设置页中展开「Selene 音乐播放器」，可以：

- 显示或隐藏播放器窗口。
- 选择启动 SillyTavern 时是否自动显示播放器。
- 选择是否在扩展菜单显示「音乐播放器」入口。
- 启用或停止系统媒体保活，并查看是否已经真正开始播放。
- 在支持 Media Session 的浏览器中，锁屏或控制中心会显示当前歌曲、歌手、专辑封面、真实时长与进度，并可播放、暂停、切换歌曲或拖动进度。
- 分别重置播放器位置与窗口尺寸。

### 系统媒体保活怎么开

可以从「Selene 音乐播放器」扩展控制栏开启，也可以在播放器中打开：

**⚙ → 系统媒体保活 → ▶ 启动保活**

开启后会自动尝试播放本地静音媒体。如果浏览器没有自动开始，扩展控制栏会显示原生音频播放条，点击其中的播放键即可；播放成功后控制条会自动隐藏。浏览器不会另外弹出“授权页面”。

显示 **系统媒体已播放** 或 **系统媒体已启动** 就代表已经开启。iPhone/iPad 切到后台后，控制中心或锁屏界面可能显示 Chrome/SillyTavern 的媒体控制；不需要时可以点击 **■ 停止保活**。

> 系统媒体保活可以降低浏览器休眠导致暂停播放或掉线的概率，但如果手机系统直接冻结或关闭浏览器，网页无法保证百分之百不断线。
