# Selene 音乐播放器

一个可通过 Git 仓库链接直接安装的 SillyTavern 第三方音乐播放器扩展。

## 功能

- 搜索和播放网络音乐
- 歌词显示与桌面歌词
- 播放队列与收藏
- 导入网易云、QQ 音乐或酷狗歌单
- 多种皮肤、迷你模式、沉浸模式
- 桌面端与移动端适配

## 安装

1. 确保运行 SillyTavern 的设备已安装 Git。
2. 打开 SillyTavern，点击顶部的「扩展」。
3. 选择「安装扩展」。
4. 粘贴本仓库的 HTTPS 链接，例如：

   ```text
   https://github.com/soso454u/-
   ```

5. 点击安装，并按 SillyTavern 的提示重新加载页面。

> 请不要粘贴 `raw.githubusercontent.com` 文件链接、ZIP 下载链接或单个 JS 文件链接；SillyTavern 需要的是完整 Git 仓库链接。

## 使用

安装后，播放器会自动显示。也可以从 SillyTavern 的扩展菜单中点击「音乐播放器」来显示或隐藏。

扩展设置页中的「Selene 音乐播放器」控制栏提供：

- 显示或隐藏播放器窗口
- 设置启动时是否自动显示
- 设置是否在扩展菜单显示「音乐播放器」入口
- 启用或停止后台音频保活
- 重置播放器位置或窗口尺寸

## 更新

扩展启用了自动更新。仓库发布新版本后，用户可在 SillyTavern 的扩展管理界面检查并安装更新。

## 数据来源与网络说明

音乐搜索、音频和歌词数据由第三方服务提供。服务的可用性、内容及所在地区的使用规则由对应服务决定。请在遵守当地法律和相关平台条款的前提下使用。

## 开发

仓库根目录必须保留以下文件：

- `manifest.json`：SillyTavern 扩展清单
- `index.js`：扩展入口
- `style.css`：扩展控制栏样式
- `纯音乐播放器.js`：播放器主体

修改后提交并推送到仓库即可发布更新。建议每次发布时同步更新 `manifest.json` 中的 `version`。
